### Tier-0 — substrate (~360 LOC)

**M1 — `HJBFiniteDifference(grid []float64, hamiltonian func(x, p, m float64) float64, viscosity float64, terminal []float64, dt, dx float64) [][]float64`.** Backward HJB-PDE `-∂_t u + H(x, ∇u, m) - σ²/2 · Δu = 0` discretized via upwind-Hamiltonian (Crandall-Lions-1992 viscosity-solution-compatible scheme): `(u^{n+1}_i - u^n_i)/Δt + H(x_i, (u^n_{i+1}-u^n_{i-1})/(2Δx), m_i) - σ²/2·(u^n_{i+1}-2u^n_i+u^n_{i-1})/Δx² = 0`. Uses upwind selection of forward / backward derivatives for monotone-discretization. Discretized matrix is **tridiagonal-banded** — uses `linalg.SolveLinearSystem` substrate. ~280 LOC. **Wholly absent.** Pin against closed-form for quadratic-Hamiltonian H(p) = p²/2 (recovers heat-equation-with-source). Reference: Crandall-Lions-1992 Bull.AMS-27, Achdou-Capuzzo-Dolcetta-2010 SIAM-J.NumerAnal-48(3), Achdou-Camilli-Capuzzo-Dolcetta-2013 §3. Cross-language pin: MathFG (Achdou-MATLAB-2010) byte-equality to 1e-8.

**M2 — `FokkerPlanckFiniteDifference(grid []float64, drift func(x, m float64) float64, viscosity float64, initial []float64, dt, dx float64) [][]float64`.** Forward FP-PDE `∂_t m - σ²/2·Δm - div(m·b(x, m)) = 0` discretized via positivity-preserving-upwind-scheme (Patankar-1980 / Larsen-Morel-Miller-1989): when drift positive uses backward-derivative on m; when negative uses forward — guarantees `m_i^n ≥ 0` for-all-n given `m_i^0 ≥ 0`. Mass-conservation pinned to 1e-12 across-time. ~280 LOC. **Wholly absent.** Pin against closed-form for OU-process drift `b = -x` (recovers Gaussian-stationary). Reference: Patankar-1980-NumericalHeatTransferFluidFlow §5, Achdou-Capuzzo-Dolcetta-2010, Carlini-Silva-2014 SIAM-J.NumerAnal-52(1).

**M3 — `LQMFGRiccati(A, B, Q, R, sigma float64, T float64) (Pfunc, ufunc func(t float64) float64)`** ⭐ **PEDAGOGICAL-MOAT.** One-dim linear-quadratic MFG closed-form via coupled-Riccati-system Bardi-Priuli-2014-DGAA: backward-Riccati `Ṗ = -2AP + B²P²/R - Q`, `P(T) = Q_T`; forward-mean `dx̄/dt = (A - B²P/R)·x̄`; forward-variance `dV/dt = 2(A - B²P/R)·V + σ²`. Returns scalar-functions of time. ~120 LOC. **Wholly absent.** Pin: with no-mean-field-coupling (Q_m = 0) recovers single-agent-LQR. Reference: Bardi-Priuli-2014 DGAA-1, Bardi-Cesaroni-2014 Math.Programming, Carmona-Delarue-2018 §6.1. **The entire pedagogical-entry-point of MFG-theory** — every textbook starts with LQ-MFG-closed-form before introducing PDE-coupling.

### Tier-1 — keystone (~1,400 LOC)

**M4 — `FiniteHorizonMFG(hamiltonian, runningCost, terminal, initial, params) MFGEquilibrium`** ⭐ **ALGORITHMIC-MOAT.** Full coupled HJB-FP-MFG-finite-horizon-solver via Newton-iteration on the coupled-system Achdou-Capuzzo-Dolcetta-2010 SIAM-J.NumerAnal-48(3): outer-loop iterates `(u^k, m^k) → (u^{k+1}, m^{k+1})` via (a) freeze m^k, solve HJB-backward for u^{k+1} using M1, (b) freeze u^{k+1}, solve FP-forward for m^{k+1} using M2, (c) check fixed-point convergence `||u^{k+1} - u^k|| + ||m^{k+1} - m^k|| < ε`. Inner-loop optionally Newton-iteration on the coupled-Jacobian (Cacace-Camilli-2012-IMAJNA-32(4)). Returns `MFGEquilibrium{Value [][]float64, Density [][]float64, Iterations int, Residual float64}`. ~440 LOC. **Wholly absent — no Go library ships this.** Pin against MathFG-Achdou-MATLAB-2010 reference-grid to 1e-6. Reference: Achdou-Capuzzo-Dolcetta-2010, Achdou-Camilli-Capuzzo-Dolcetta-2013-SIAM-J.NumerAnal-51(5).

**M5 — `ErgodicMFG(hamiltonian, runningCost, viscosity, params) (lambda float64, value, density []float64)`.** Stationary-MFG `λ + H(x, ∇u, m) - σ²/2·Δu = 0` paired with `-σ²/2·Δm - div(m·∇_p H) = 0` and `∫m = 1`. Solves via fixed-point on (λ, u, m): outer loop adjusts λ via secant-method, inner loop solves HJB for u given (λ, m) and FP for m given u. Lasry-Lions-2007 ergodicity-condition guarantees existence-uniqueness under monotonicity. ~360 LOC. **Wholly absent.** Pin: with quadratic-Hamiltonian recovers closed-form λ = (σ² log(2π))/2 + cost-rate. Reference: Lasry-Lions-2007 Jpn.J.Math-2(1), Cardaliaguet-2010-NotesPDE-CollegedeFrance §4.

**M6 — `FictitiousPlayMFG(hamiltonian, runningCost, terminal, initial, params, maxIter int) MFGEquilibrium`.** Iterative best-response with averaged-distribution Cardaliaguet-Hadikhanloo-2017 ESAIM:COCV-23(2): at iteration k, set `m̄^k = (1/k) Σ_{j=1}^k m^j`, solve HJB given `m̄^k` for `u^{k+1}`, derive optimal-control α^{k+1}, propagate FP under α^{k+1} to get m^{k+1}. Converges to MFG-equilibrium under Lasry-Lions-monotonicity. ~280 LOC. **Wholly absent.** Pin: convergence-rate `||m^k - m^*|| = O(1/√k)` matches theory. Reference: Cardaliaguet-Hadikhanloo-2017, Hadikhanloo-Silva-2019-ESAIM.

**M7 — `LasryLionsMonotonicityCheck(F func(x, m float64) float64, m1, m2 []float64) bool`.** Property-based test for monotonicity condition `∫(F(x, m_1) - F(x, m_2))·(m_1 - m_2) dx ≥ 0`. Critical golden-file pin on the existence-uniqueness condition. ~80 LOC. **Wholly absent.** Pin: holds for `F(x, m) = ∫|x-y|·m(y)dy` (cost-of-density-overlap); fails for `F(x, m) = -ρ(m)` (negative-density-coupling). Reference: Lasry-Lions-2007 Theorem-2.5.

**M8 — `PropagationOfChaos(N int, T float64, drift func(x, m []float64) float64, sigma float64, x0sample []float64) (particles [][][]float64, w2distance []float64)`.** N-particle-system simulation via Euler-Maruyama on McKean-Vlasov SDE `dX^i_t = b(X^i_t, μ^N_t)·dt + σ·dW^i_t` where `μ^N_t = (1/N)Σ δ_{X^j_t}`. Compares empirical-distribution to MFG-density via W_2-Wasserstein (cross-link 201 OT-Sinkhorn) at each time step. Verifies convergence-rate Sznitman-1991 / Cardaliaguet-2017 `W_2(μ^N, m) = O(1/√N)`. ~280 LOC. **Wholly absent — BLOCKED on 117-Box-Muller-keystone and 202-Brownian-path.** Pin: empirical-W_2-vs-MFG converges as 1/√N to within MC standard-error. Reference: Sznitman-1991 St.Flour-1464 §1, Cardaliaguet-2017 Ann.Probab-45(1), Lacker-2020 Ann.Probab-48(5).

## 4. Risks and gotchas

- **G1. CFL-restriction of explicit-FD-MFG.** Explicit-time-stepping on M1/M2 requires `Δt ≤ Δx²/(2σ²)` for stability. Document this; recommend semi-Lagrangian (M10) or implicit-time-stepping for production with large σ. Without this, naive explicit-FD on small-noise (σ → 0) requires Δt → 0² which is intractable.
- **G2. Newton-iteration divergence on M4.** Coupled-system Jacobian `[∂HJB/∂u ∂HJB/∂m; ∂FP/∂u ∂FP/∂m]` is non-symmetric and can be ill-conditioned near MFG-equilibrium with weak-monotonicity. Document loud-fail when condition-number exceeds threshold; provide fictitious-play-fallback (M6) as numerically-robust alternative.
- **G3. Non-uniqueness without monotonicity.** Lasry-Lions-monotonicity (M7) is sufficient but not necessary for uniqueness. Without it, M4/M5 can converge to different equilibria from different initializations. Document loud-fail when M7-test fails and recommend running M4 from multiple initializations to verify equilibrium-set.
- **G4. McKean-Vlasov-particle-system bias.** N-particle approximation has propagation-of-chaos rate `O(1/√N)` — for production-MFG-from-N-particles, need N ≥ 1000 to get 3% relative error. Document this; recommend M4-coupled-PDE-solver as the production path and M8/M12 as validation-only.
- **G5. LQ-MFG closed-form requires linearity.** M3/M13 require `H, F, g` quadratic in (x, p, m). Document loud-fail when input violates linearity; suggest M4 finite-horizon for non-linear cases.
- **G6. Master-equation curse-of-dimensionality.** M11 master-equation lives on `P(R^d)` which is infinite-dimensional. Finite-state discretization (state-space size |S|) reduces to PDE on simplex of dim |S|-1, but |S|=10 already requires 10⁴-grid for reasonable accuracy. Document this; ship only |S| ≤ 5 in production.
- **G7. Common-noise vs idiosyncratic-noise distinction.** M16 distinguishes `dW^0` (common, affects all) vs `dW^i` (idiosyncratic, independent across agents). Mishandling this in particle-simulation gives wrong-MFG-equilibrium. Document explicitly with a code-comment plus a property-test pin.
- **G8. Cross-language byte-equality requires fixed grid + scheme.** MathFG (Achdou-MATLAB) uses upwind-Engquist-Osher Hamiltonian-discretization at corners; reality must match this exact-formula or pin will fail. Document explicitly the upwind-corner-handling and pin path-grid in golden-file fixtures.
