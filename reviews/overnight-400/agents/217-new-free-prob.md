## 1. The sixteen free-probability primitives

Each entry: capability + reference / composition / LOC / cross-link / blocking-flag.

**F1 — `CauchyTransform(measure []float64, weights []float64, z complex128) complex128` / `StieltjesPerronInvert(G func(complex128) complex128, lambda float64, eps float64) float64`.** Cauchy transform `G_μ(z) = ∫ dμ(λ)/(z − λ)` analytic on upper-half-plane Im z > 0. Discrete-measure forward `Σ_i w_i / (z − x_i)`. Inversion: evaluate `G(λ + iε)` for small ε and take `−Im/π`. Pure scalar formula on complex inputs. ~140 LOC. **Identical to 216-R6 (subset); ship once via 216-R6.** Pin against `G_{μ_sc}(z) = (z − √(z² − 4))/2` for z = 5 + i to 1e-12.

**F2 — `RTransform(measure []float64, weights []float64, z complex128) complex128` / `STransform(...) complex128` / `RTransformAdd(R1, R2 func(complex128) complex128) func(complex128) complex128`.** Voiculescu-1986 R-transform `R_μ(z) = G_μ^{-1}(z) − 1/z` and Voiculescu-1987 S-transform `S_μ(z) = (1+z)/z · ψ_μ^{-1}(z)`. R linearizes ⊞: `R_{μ⊞ν}(z) = R_μ(z) + R_ν(z)`. S linearizes ⊠: `S_{μ⊠ν}(z) = S_μ(z) · S_ν(z)`. Numerical inversion of `G_μ` via complex-Newton's-method on upper-half-plane (single-step `z_{k+1} = z_k − (G(z_k) − w)/G'(z_k)`). ~280 LOC. **Identical to 216-R7; ship once via 216-R7.** Pin: `R_{μ_sc}(z) = z` (semicircle has linear R-transform), `R_{free-Poisson(λ=1)}(z) = 1/(1-z)`, `R_{δ_a + δ_b}` arithmetic.

**F3 — `SemicirclePDF/CDF/Support(x, sigma2) float64` / `SemicircleR(z complex128) complex128 = z·sigma2`.** Wigner-1955 semicircle `dμ_sc = (1/(2π σ²))·√(4σ² − x²)·1_{|x|≤2σ} dx` plus its R-transform `R(z) = σ²·z` (the LINEAR R-transform — semicircle is to free-prob what Gaussian is to classical-prob). Pure scalar. ~80 LOC. **Identical to 216-R15 (PDF/CDF only); F3 ADDS R-transform identity** which 216 did not surface. Ship the R(z)=σ²z identity HERE but the PDF/CDF in 216-R15.

**F4 — `NonCrossingPartitions(n int) [][][]int` / `IsNonCrossing(partition [][]int) bool` / `CountNonCrossing(n int) int = CatalanNumber(n)`.** Enumerate set-partitions of `[n] = {1,...,n}` whose blocks have no crossings: there exist no `i < j < k < l` with `i, k ∈ B_1`, `j, l ∈ B_2`, `B_1 ≠ B_2`. Cardinality `|NC(n)| = C_n` (Catalan). Bijection to Dyck-paths. Generate by filtering all-set-partitions (BLOCKED on 037-combinatorics-missing `SetPartitions`) OR by direct ECO-recursive-construction (Flajolet-Sedgewick-2009). ~140 LOC. **Wholly absent from 216 (RMT slot did not surface combinatorial foundation).** Pin `len(NonCrossingPartitions(n)) == int(CatalanNumber(n))` for n=1..10. Ship in `combinatorics/noncrossing.go`.

**F5 — `FreeCumulants(moments []float64) []float64` / `MomentsFromFreeCumulants(kappas []float64) []float64`.** Speicher-1994 moment-cumulant bijection on NC(n): `m_n = Σ_{π ∈ NC(n)} κ_π` and inverse via Möbius `κ_n = Σ_{π ∈ NC(n)} μ_NC(0_π, 1_π) · m_π` with `μ_NC(0, 1_n) = (-1)^{n-1}·C_{n-1}`. Computes free cumulants from moments and vice versa. Critical: classical cumulants use Π(n) (Bell-many), free cumulants use NC(n) (Catalan-many) — same machinery on different lattice. Composes F4 + F6. ~180 LOC. **Wholly absent from 216.** Pin against semicircle: `κ_n(μ_sc) = δ_{n,2}·σ²` (free-Gaussian has only κ_2); free-Poisson with rate λ: `κ_n = λ`. Ship in `prob/freeprob/cumulants.go`.

**F6 — `KrewerasComplement(partition [][]int, n int) [][]int` / `NCMobius(pi, sigma [][]int) int`.** Kreweras-1972 Discrete-Math-1 complementation map K: NC(n) → NC(n) involution via interleaved-points-in-NC(2n) construction. Möbius function on NC-lattice `μ_NC(0, π) = Π_blocks (-1)^{|B|-1}·C_{|B|-1}` (signed product of Catalan). Used in F5 cumulant inversion. Both pure-combinatorial. ~120 LOC. **Wholly absent from 216.** Pin Kreweras-bijection-property `K∘K = id`; `μ_NC(0, 1_3) = -2 = -(C_2)` etc. Ship in `combinatorics/noncrossing.go`.

**F7 — `FreePoissonPDF/CDF/Support(x, lambda, sigma2) float64` / `FreePoissonR(z complex128) complex128`.** Free Poisson distribution = Marchenko-Pastur with parameter `λ` and atomic mass `(1−λ)δ_0` for `λ < 1`. R-transform `R(z) = λ/(1 − z)`. Compound free Poisson `μ_{λ, ν} = lim_{n→∞} ((1 − λ/n)·δ_0 + (λ/n)·ν)^{⊞n}`. Density `f(x) = √((λ_+ − x)(x − λ_−))/(2π·σ²·x)` with edges `λ_± = σ²·(1±√λ)²`. ~120 LOC. **PARTIAL OVERLAP 216-R1 (MP-density-formula only); F7 ADDS R-transform identity AND free-Poisson interpretation as compound limit.** Pin `R_{free-Poisson(λ)}(z) = λ/(1-z)`; coincides with R-transform of geometric-classical-Poisson under Bercovici-Pata bijection F10.

**F8 — `FreeAdditiveConvolutionSubordination(mu, nu [](sample, weight), grid []float64) []float64`.** Belinschi-Mai-Speicher-2017 J.Funct.Anal-273 subordination-fixed-point algorithm for `μ ⊞ ν`. Joint subordination `G_{μ⊞ν}(z) = G_μ(ω_1(z)) = G_ν(ω_2(z))` with `ω_1(z) + ω_2(z) − z = 1/G_{μ⊞ν}(z)`. Numerical algorithm: fixed-point iteration on the system `ω_1 = z + 1/G_ν(ω_2) − 1/G_{μ⊞ν}`, `ω_2 = z + 1/G_μ(ω_1) − 1/G_{μ⊞ν}` with stable contraction-to-fixed-point. Inverse-density via Stieltjes-Perron F1. Composes F1 + F2. ~280 LOC. **STRICT SUPERSET of 216-R7 numerical-method;** R-transform addition (F2 inversion) is the slow algebraic-manipulation; subordination is the production-numerical-algorithm. Pin against semicircle ⊞ semicircle = scaled-semicircle radius √2 to 1e-6 in Wasserstein-1.

**F9 — `FreeMultiplicativeConvolutionSubordination(mu, nu, grid []float64) []float64`.** Belinschi-Bercovici-2007-J.Funct.Anal-247 multiplicative subordination for `μ ⊠ ν`. **F10 — `BercoviciPataBijection(classical_distribution string, params []float64) string` / `ClassicalToFreeMoments(moments []float64) []float64`.** Bercovici-Pata-1999 Ann.Math-149 bijection between classical-infinitely-divisible-measures and free-infinitely-divisible-measures. **F11 — `FreeCLT(samples [][]float64, var_target float64) (statistic, pvalue float64)`.** Voiculescu-1985 free-CLT: `(a_1+...+a_n)/√n → s` (semicircular) for free identically-distributed mean-zero unit-variance. ## 5. Precision hazards

F1 Cauchy-transform: numerical evaluation near real-axis (Im z → 0+) suffers from cancellation; require Im z ≥ 1e-3 in production. Stieltjes-Perron inversion `f_μ(λ) = -lim_{ε↓0} Im G/π` introduces O(ε) bias — choose ε via L-curve heuristic. F2 R-transform inversion `G_μ^{-1}` may have multiple branches on positive-imaginary half-plane — guard with branch-cut handling and Newton-step monotonicity check. F4 Non-crossing partition enumeration: `C_30 ≈ 3.8e15` exceeds float64 precision in CatalanNumber; switch to math/big for n ≥ 30 enumeration counting (but the actual partition list itself fits in O(C_n) memory only for n ≤ 14 or so). F5 Möbius-inversion: high-order moments computation via NC enumeration is O(C_n) memory and time — practical limit n ≤ 12 (C_12 ≈ 208,012 partitions). F6 Kreweras-complement: bijection only on NC(n); applying to crossing partitions silently gives garbage — guard with IsNonCrossing check. F7 Free-Poisson with λ ≤ 1 has atom at zero — `(1−λ)·δ_0 + (1/λ)·MP-density` mixed measure handling. F8 Subordination fixed-point: divergence near singular support of input measures; require regularization parameter ε to match F1 inversion ε. F8 Belinschi-Mai-Speicher-2017 contraction-property only proven on upper-half-plane Im z > 0 — initialization and iteration must stay strictly in upper-half-plane. F9 Multiplicative subordination: positivity-of-spectrum required; both inputs must be supported on R_+ for ⊠ definition to make sense. F10 Bercovici-Pata: bijection is at level of moments via Lévy-Khintchine triplet; finite-moment-truncation introduces bias for heavy-tailed (α < 2 stable) cases. F11 Free-CLT empirical: convergence rate is O(1/√n) in Wasserstein — need n ≥ 100 free samples for chi² p-value > 0.05. F12 Asymptotic-freeness: convergence rate O(1/n) on each mixed moment — n=200 matrices give relative error ~5% on fourth-moment. F13 Circular element: spectral distribution = uniform on disk radius 1, NOT circular law for sum of free Haar unitaries (which approaches uniform on disk radius √n). F14 Free Fisher: integrand `p(x)³` requires absolutely-continuous distribution — atomic measures give +∞. F15 Free entropy: `∫∫ log|x-y|·p(x)·p(y)` is the Voiculescu logarithmic-energy — equivalent to negative-electrostatic-energy of charge distribution; numerically integrate via 2D Gauss-Legendre with regularization on x=y diagonal. F16 Operator-valued: B-valued analytic functions are non-commutative — direct port of scalar algorithms incorrect.
