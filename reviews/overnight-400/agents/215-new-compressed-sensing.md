### Tier 1 — Sensing matrices + soundness certificates (~520 LOC, substrate)

**C1 — `cs/sensing/sensing.go` ~200 LOC.** `Gaussian(m, n, seed) []float64` row-major iid `N(0, 1/m)` matrix (column-normalized variant `GaussianNormalized`), `Bernoulli(m, n, seed) []float64` iid `±1/√m`, `Rademacher(m, n, seed)` (alias), `PartialFourier(m, n, indices [m]int) []complex128` random m rows of n×n DFT matrix (depends on signal/fft for FFT-based MatVec when `n` is large), `Hadamard(m, n, indices) []float64` random m rows of Sylvester-Hadamard matrix (n must be power of 2), `Toeplitz` and `Circulant` (Romberg-2009 structured-CS), `STM` Spread-Spectrum-Modulation. Depends on **NEW `prob/random/normal.go`** (~80 LOC) `RandNormal(seed *Source, n int) []float64` Box-Muller or Marsaglia-polar; `RandBernoulli(seed *Source, n int, p float64) []bool`; deterministic-by-seed for golden-file reproducibility. Refs: Candes-Romberg-Tao-2006 *Robust uncertainty principles* §III; Baraniuk-Davenport-DeVore-Wakin-2008 *Simple proof of RIP for random matrices*; Foucart-Rauhut 2013 §9.

**C2 — `cs/sensing/coherence.go` ~80 LOC.** `MutualCoherence(A []float64, m, n int) float64` returns `μ(A) = max_{i≠j} |⟨a_i, a_j⟩|/(||a_i||·||a_j||)` in O(mn^2) (column-normalize then max-abs-off-diag of A^T·A). `WelchBound(m, n int) float64 = sqrt((n-m)/(m*(n-1)))` lower bound on achievable μ. `BabelFunction(A, k int) float64` = Tropp-2004 Babel-function `μ_1(k) = max_{|S|=k} max_{i∉S} Σ_{j∈S}|⟨a_i,a_j⟩|/(||a_i||·||a_j||)` — refines coherence for OMP-soundness `μ_1(s-1) + μ_1(s) < 1`. `CoherenceSparsityBound(μ float64) int = floor((1 + 1/μ)/2)` Donoho-Elad-2003 uniqueness threshold. Refs: Donoho-Huo-2001 *Uncertainty principles and ideal atomic decomposition*; Tropp-2004 *Greed is good*; Welch-1974.

**C3 — `cs/sensing/rip.go` ~120 LOC.** `EmpiricalRIPConstant(A []float64, m, n, s, numTrials int, seed *Source) float64` Monte-Carlo estimator: sample numTrials s-sparse unit-vectors x with random support and signs, return `max_x | ||Ax||_2^2 - ||x||_2^2 |` lower-bound on δ_s (true δ_s is NP-hard to compute, Bandeira-Dobriban-Mixon-Sawin-2013 — but Monte-Carlo gives certifiable lower bound). `RIPCertViaCoherence(μ, s) float64 = (s-1)*μ` Gershgorin-bound on δ_s via coherence (loose but cheap). `RIPCertViaSpark(spark, s) bool = spark > 2*s` uniqueness from spark. `BPSufficientCondition(δ_2s float64) bool = δ_2s < 0.4931` Cai-Wang-Xu-2010 sharp BP-recovery condition. `OMPSufficientCondition(μ, s) bool = μ < 1/(2*s-1)` Tropp-2004 ERC. `IHTSufficientCondition(δ_3s) bool = δ_3s < 1/sqrt(32)` Blumensath-Davies-2008. Refs: Candes-Tao-2005 *Decoding by linear programming* RIP-original; Cai-Wang-Xu-2010 sharp threshold; Foucart-Rauhut 2013 §6.

**C4 — `cs/sensing/spark.go` ~120 LOC.** `Spark(A, m, n int) int` brute-force minimum number of linearly-dependent columns (NP-hard in general; correct for `n ≤ 30` enumeration up to C(n, k) for k = 2, 3, ..., n). `KruskalRank(A, m, n int) int = spark(A) - 1`. `NSPVerify(A, m, n, s int, k_random_h int, seed *Source) (held bool, witness []float64)` Monte-Carlo NSP-check: sample k_random_h null-space vectors via SVD-null-space + random-combination, check `||h_S||_1 < ||h_{S^c}||_1` for all `|S| ≤ s`; returns first counterexample if NSP fails. Depends on `linalg/svd` for null-space. Refs: Donoho-Elad-2003 *Optimally sparse representation* spark-original; Cohen-Dahmen-DeVore-2009 NSP-vs-RIP relationship.

### Tier 2 — Greedy pursuit (~720 LOC)

**C5 — `cs/greedy/omp.go` ~120 LOC — KEYSTONE.** `OMP(A, y, sparsity int, maxIter int) (x []float64, support []int, residNorm float64, iters int)`. Algorithm: r₀ = y, S₀ = ∅; at iter k pick `j_k = argmax_j |a_j^T r_{k-1}|`, append to support, x_S = (A_S^T A_S)⁻¹ A_S^T y via QR-update (linalg.QR), r_k = y - A_S x_S; stop when k = sparsity OR ||r||_2 < tol. O(s·m·n) total. Equivalence: Cai-Wang-2011 OMP recovers s-sparse exactly under RIP-(s+1) with `δ_{s+1} < 1/(3·sqrt(s))`. Refs: Pati-Rezaiifar-Krishnaprasad-1993; Tropp-Gilbert-2007 *Signal recovery from random measurements via OMP*; Cai-Wang-2011 sharp RIP-condition.

**C6 — `cs/greedy/cosamp.go` ~180 LOC.** `CoSaMP(A, y, s int, maxIter int) (x []float64, residNorm float64, iters int)`. Algorithm at iter k: (1) Identify: pick top-2s indices of `A^T r_{k-1}` magnitude, (2) Merge: T = support(x_{k-1}) ∪ identified, (3) Estimate: solve LS on T, (4) Prune: keep top-s of LS-result. O(s·m·n) per iter; provably converges in O(log(||x||_2/η)) iters under RIP-4s ≤ 0.1. Refs: Needell-Tropp-2008 *CoSaMP: Iterative signal recovery from incomplete and inaccurate samples*; Foucart-2011 *Hard thresholding pursuit* §4.

**C7 — `cs/greedy/iht.go` ~120 LOC — KEYSTONE.** `IHT(A, y, s int, stepSize float64, maxIter int) (x []float64, residNorm float64, iters int)`. Algorithm: x₀ = 0; iterate `x_{k+1} = H_s(x_k + μ A^T (y − A x_k))` where H_s is hard-threshold-keep-top-s (= ProxL0 truncated to support-size-s, NOT ProxL0 with γ-threshold which is the existing `optim/proximal/operators.go:48` operator; needs new `HardThresholdTopS(v, s, out)` ~10 LOC). `NIHT` (Normalized-IHT) Blumensath-Davies-2010 adapts μ per-iter via `μ_k = ||A_S^T r||^2 / ||A A_S^T r||^2` for stability without needing ||A||_2 < 1 normalization. `AcceleratedIHT` Nesterov-momentum variant. Refs: Blumensath-Davies-2008 *Iterative thresholding for sparse approximations*; Blumensath-Davies-2009 *Iterative hard thresholding for compressed sensing* + 2010 *Normalized iterative hard thresholding*.

**C8 — `cs/greedy/htp.go` ~80 LOC.** `HTP(A, y, s int, stepSize float64, maxIter int)` Hard-Thresholding-Pursuit. Like IHT but after thresholding solves LS exactly on the support (combines IHT-step with OMP-style support-fix-and-LS). Foucart-2011 proves HTP requires fewer iterations than IHT under same RIP. Refs: Foucart-2011 *Hard thresholding pursuit: an algorithm for compressive sensing*.

**C9 — `cs/greedy/sp.go` ~80 LOC.** `SubspacePursuit(A, y, s int, maxIter int)` Dai-Milenkovic-2009. Like CoSaMP but identifies only s indices per iter (not 2s) and uses `T = identified ∪ support(x_{k-1})` then prunes to s after LS. Refs: Dai-Milenkovic-2009 *Subspace pursuit for compressive sensing signal reconstruction*.

**C10 — `cs/greedy/stomp.go` ~80 LOC.** `StagewiseOMP(A, y, threshold float64, maxIter int)` Donoho-Tsaig-Drori-Starck-2012. Adds ALL coordinates above threshold-times-noise-level per iter (not just one); much faster than OMP for highly-structured signals. `RegularizedOMP` Needell-Vershynin-2009 variant. Refs: Donoho-Tsaig-Drori-Starck-2012 *Sparse solution of underdetermined systems by stagewise OMP*; Needell-Vershynin-2009 *Uniform uncertainty principle and signal recovery via ROMP*.

**C11 — `cs/greedy/wmp.go` ~60 LOC.** `WeakOMP(A, y, s int, weakness float64)` Temlyakov-2000 weak greedy: relaxes argmax to "any column within (1-α) of argmax magnitude" for parallel-friendly variants. `MatchingPursuit` (non-orthogonal-MP, Mallat-Zhang-1993) included for completeness — strictly inferior to OMP but historically important. Refs: Mallat-Zhang-1993 *Matching pursuits with time-frequency dictionaries*; Temlyakov-2000.

### Tier 3 — Convex relaxation (~640 LOC)

**C12 — `cs/convex/bp.go` ~120 LOC.** `BasisPursuit(A, y, m, n) (x []float64)` solves `min ||x||_1 s.t. Ax = y` via primal-dual interior-point (Chen-Donoho-Saunders-1998 PDIPM) OR via ADMM consensus formulation `min ||x||_1 + I_{Ax=y}(z) s.t. x = z` reusing existing `optim/proximal/Admm` with `proxF = ProxL1` and `proxG = projection-onto-affine-subspace-{z: Az = y}` (~40 LOC for the affine-projection prox via QR-pseudoinverse). Affine-projection-prox is the same operator that `cs/sensing/spark.go` uses. Refs: Chen-Donoho-Saunders-1998 *Atomic decomposition by basis pursuit*.

**C13 — `cs/convex/bpdn_admm.go` ~120 LOC — KEYSTONE.** `BPDN(A, y, sigma, rho, maxIter)` solves `min ||x||_1 s.t. ||Ax - y||_2 ≤ σ` via ADMM splitting `min ||x||_1 + I_{||Az-y||≤σ}(z) s.t. x = z`. The z-update is projection-onto-L2-ball-around-y-of-radius-σ-after-affine-transform-A — no closed form so falls back to LASSO-Lagrangian-equivalent for σ-tuning via SPGL1-VanDenBerg-Friedlander-2008 root-finding on `||Ax(λ) - y||_2 - σ`. Refs: Chen-Donoho-Saunders-1998 (BPDN-original); Van-Den-Berg-Friedlander-2008 *Probing the Pareto frontier for basis pursuit solutions* (SPGL1).

**C14 — `cs/convex/lars.go` ~180 LOC.** `LARS(A, y, n int) (path []LARSStep)` Efron-Hastie-Johnstone-Tibshirani-2004 LASSO regularization-path computed in closed-form piecewise-linear-in-λ — every breakpoint is when a coordinate enters/leaves the active set. O(n^3 + n m s) for the entire path. Returns slice of {λ, x, support}; consumer picks any λ-cross-validation point. Reuses `linalg/QR` for active-set-update. Refs: Efron-Hastie-Johnstone-Tibshirani-2004 *Least angle regression* AOS 32:407.

**C15 — `cs/convex/dantzig.go` ~80 LOC.** `DantzigSelector(A, y, lambda)` solves `min ||x||_1 s.t. ||A^T(y - Ax)||_∞ ≤ λ` via LP-feasibility (interior-point on the dual). Near-oracle MSE bound `||x̂ − x||_2^2 ≤ C·s·σ^2·log(n)` matches LASSO up to constants but with simpler theoretical-analysis. Refs: Candes-Tao-2007 *The Dantzig selector: statistical estimation when p is much larger than n*.

**C16 — `cs/convex/reweighted.go` ~80 LOC.** `ReweightedL1(A, y, lambda, numIter int)` Candes-Wakin-Boyd-2008: iterate `x_{k+1} = argmin Σ_j w_j^k |x_j| s.t. ||Ax-y|| ≤ σ` with `w_j^k = 1/(|x_j^k| + ε)`. Equivalent to MM-algorithm for log-sum-penalty `Σ log(|x_j| + ε)` which is concave in |x_j| → strictly sharper sparsity than L1. Each iter calls C12 BasisPursuit or C13 BPDN. Refs: Candes-Wakin-Boyd-2008 *Enhancing sparsity by reweighted l1 minimization*.

**C17 — `cs/convex/elastic.go` ~60 LOC.** `ElasticNet(A, y, lambda1, lambda2)` Zou-Hastie-2005 `min ||Ax-y||_2^2 + λ_1 ||x||_1 + λ_2 ||x||_2^2` — handles correlated-predictors better than LASSO, group-pickup-effect. Reuses existing `optim/proximal/Fbs` with `prox = soft-threshold-on-(λ_1·step) + scale-by-1/(1+2·λ_2·step)` (closed-form for L1+L2-prox, no algorithmic complexity beyond LASSO). Refs: Zou-Hastie-2005 *Regularization and variable selection via the elastic net*.

### Tier 4 — Approximate Message Passing frontier (~720 LOC)

**C18 — `cs/amp/amp.go` ~280 LOC — SINGULAR-MOAT-KEYSTONE.** `AMP(A, y, denoiser Denoiser, maxIter int) (x []float64, mse []float64)` Donoho-Maleki-Montanari-2009. Algorithm: x₀ = 0, r₀ = y; at iter k:
```
x_{k+1} = denoiser(x_k + A^T r_k; θ_k)
b_{k+1} = (1/m) · Σ denoiser'(x_k + A^T r_k; θ_k)        [Onsager correction]
r_{k+1} = y − A x_{k+1} + b_{k+1} · r_k
θ_{k+1} = update_threshold(||r_{k+1}||_2 / sqrt(m))     [State-Evolution]
```
The Onsager-correction `b_k · r_k` is what distinguishes AMP from ISTA — it makes the residual `A^T r_k` behave EXACTLY like iid-Gaussian-noise with variance σ_k^2 = State-Evolution-output. Denoiser interface: `type Denoiser interface { Apply(in []float64, theta float64, out []float64); Derivative(in []float64, theta float64) float64 }`. Built-in denoisers in C19. State-Evolution-recursion `σ_{k+1}^2 = σ_w^2 + (1/δ) E[ (η(X + σ_k Z; θ_k) - X)^2 ]` predicts MSE-trajectory exactly in n→∞ limit, allows AMP to be **calibrated** without cross-validation. Refs: Donoho-Maleki-Montanari-2009 *Message-passing algorithms for compressed sensing* PNAS 106:18914 (the foundational paper); Maleki-Montanari-2010 *State evolution for the random graphical model*; Bayati-Montanari-2011 *The dynamics of message passing on dense graphs* for SE-rigorous-proof.

**C19 — `cs/amp/denoisers.go` ~160 LOC.** Library of scalar denoisers η(x; θ): `SoftThreshold(x, θ)` = sign(x)·max(|x|-θ, 0) (= ProxL1 elementwise, the original AMP-DMM denoiser); `HardThreshold(x, θ)` = x·1{|x|>θ}; `MMSE_BernoulliGaussian(x, θ; ε, σ_x)` = E[X | X+Z=x] under prior `X ~ ε·N(0, σ_x^2) + (1-ε)·δ_0` (closed-form via Bayes); `SUREShrink(x, θ)` = Stein-Unbiased-Risk-Estimator-optimal shrinker; `MMSE_Bayesian(x, θ; prior)` generic via numerical-integration. Refs: Donoho-Johnstone-1995 *Adapting to unknown smoothness via wavelet shrinkage* SURE; Maleki-2009 PhD thesis ch.3 Bayesian-AMP-denoisers.

**C20 — `cs/amp/state_evolution.go` ~120 LOC.** `StateEvolution(prior Prior, denoiser Denoiser, delta, sigma_w float64, maxIter int) []float64` returns MSE-sequence `{σ_0^2, σ_1^2, ..., σ_K^2}` via fixed-point recursion `σ_{k+1}^2 = σ_w^2 + (1/δ) E[(η(X + σ_k Z; θ_k) - X)^2]` where E is over `X ~ prior, Z ~ N(0,1)`. Numerical-integration via Gauss-Hermite-quadrature for general priors. Phase-transition curve `δ*(ε) = inf{δ : SE-fixed-point at zero MSE}` = Donoho-Maleki-Montanari-2009 minimax-MSE curve which matches the BP `l_1`-vs-`l_0` phase-transition curve to machine precision. Refs: Bayati-Montanari-2011 SE-rigorous-proof; Donoho-Tanner-2010 *Counting faces of randomly projected polytopes* phase-transition-comparison.

**C21 — `cs/amp/vamp.go` ~120 LOC.** `VAMP(A, y, denoiser, maxIter)` Rangan-Schniter-Fletcher-2017. Right-rotationally-invariant A (so AMP-State-Evolution matches even for non-iid-A), MMSE-denoiser-per-iter, more robust than AMP under "bad" matrices (e.g., column-correlated, ill-conditioned). Algorithm uses LMMSE-estimator on linear-stage + scalar-MMSE-denoiser on nonlinear-stage with two-step messages. Refs: Rangan-Schniter-Fletcher-2017 *Vector approximate message passing*.

**C22 — `cs/amp/gamp.go` ~160 LOC.** `GAMP(A, y, output_q, denoiser, maxIter)` Generalized-AMP-Rangan-2011 handles non-Gaussian outputs `y_i = q(z_i, w_i)` (binary-classification: `y = sign(z + w)` for 1-bit-CS; Poisson-regression: `y ~ Poisson(exp(z))` for low-photon-imaging; PR: `y = |z|^2 + w` for phase-retrieval). Output-channel-likelihood-derivative `g(z, y, σ^2)` and denoiser-MMSE `η(s, σ^2)` are user-supplied. Refs: Rangan-2011 *Generalized approximate message passing for estimation with random linear mixing*; Schniter-Rangan-2015 *Compressive phase retrieval via generalized approximate message passing* (PR-GAMP).

### Tier 5 — Low-rank recovery (~520 LOC, depends on linalg/svd)

**C23 — `cs/lowrank/svt.go` ~80 LOC.** `SingularValueThresholding(M, tau float64, out)` applies `D_τ(M) = U·max(Σ-τ, 0)·V^T` — proximal-operator of `τ·||·||_*` (nuclear-norm). Building-block of SVT-iteration `X_{k+1} = D_τ(X_k - β·(P_Ω(X_k) - P_Ω(M)))` for matrix-completion (Cai-Candes-Shen-2010). Refs: Cai-Candes-Shen-2010 *A singular value thresholding algorithm for matrix completion*. **Depends on `linalg/svd`.**

**C24 — `cs/lowrank/matrix_completion.go` ~140 LOC.** `MatrixCompletionSVT(observed map[[2]int]float64, m, n, rank, tau, maxIter int) (X []float64)` SVT-based; `MatrixCompletionADMM` augmented-Lagrangian variant (faster convergence per Lin-Chen-Ma-2010 ALM); `MatrixCompletionIRLS` (Mohan-Fazel-2012 reweighted-least-squares-for-rank). Sample-complexity `m_obs ≥ C·n^{1.2}·r·log(n)` per Candes-Recht-2009. Refs: Candes-Recht-2009 *Exact matrix completion via convex optimization*; Recht-2011 sharper-bound; Lin-Chen-Ma-2010 ALM.

**C25 — `cs/lowrank/robust_pca.go` ~180 LOC.** `RobustPCA(M, lambda float64, maxIter) (L, S [][]float64)` Principal-Component-Pursuit-Candes-Li-Ma-Wright-2011 solves `min ||L||_* + λ·||S||_1 s.t. M = L + S` via ADMM: L-step is SVT(M - S + Y/μ; 1/μ), S-step is ProxL1(M - L + Y/μ; λ/μ), Y-update is dual-ascent. λ = 1/sqrt(max(m,n)) the universal-balance per CLMW-2011 Thm 1.1. `StableRobustPCA` (noise-tolerant: M = L + S + N) Zhou-Li-Wright-Candes-Ma-2010. Refs: Candes-Li-Ma-Wright-2011 *Robust principal component analysis?* JACM 58:11.

**C26 — `cs/lowrank/iht_matrix.go` ~120 LOC.** `MatrixIHT(M, rank, stepSize, maxIter int)` matrix-version of IHT: hard-thresholding via truncated-SVD (keep top-r singular values); used for direct rank-constrained matrix-completion when nuclear-norm is too slow. `RIHT` (Riemannian-IHT-Wei-Cai-Chan-Leung-2016) on Stiefel-manifold for orthogonal-constraint variants. Refs: Goldfarb-Ma-2011 *Convergence of fixed-point continuation algorithms for matrix rank minimization*; Wei-Cai-Chan-Leung-2016 Riemannian-IHT.
