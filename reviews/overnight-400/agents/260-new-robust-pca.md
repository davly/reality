## 1. The twenty-six primitives R1–R26

Tier numbering: T1 = ProxNuclear+norm-substrate, T2 = PCP / Inexact-ALM / Accelerated, T3 = IRLS / Schatten-p / IRNN, T4 = GoDec / Stable-PCP, T5 = Online / streaming, T6 = L1-PCA / M-estimator, T7 = Tensor / Bayesian / Variational / Dictionary.

### Tier-1 — ProxNuclear + nuclear/Schatten/operator/balance/incoherence (~140 LOC NEW)

**R1 — `optim/proximal/operators.go::ProxNuclear` ~40 LOC.** `ProxNuclear(in []float64, m, n int, gamma float64, out []float64)` applies `D_γ(M) = U·max(Σ−γ, 0)·V^T` — proximal-operator of `γ·||·||_*`. Building-block of EVERY RPCA solver. Calls `linalg.SVD` once per invocation. Closes `optim/proximal/doc.go:62` promise-debt. **IDENTICAL to 259-M1, ship ONCE.** Refs: Cai-Candes-Shen-2010-SIOPT-20:1956 *A singular value thresholding algorithm for matrix completion*.

**R2 — `linalg/nuclear.go::NuclearNorm + SchattenP + OperatorNorm + StableRank` ~50 LOC.** SVD-derived. `StableRank(M) = ||M||_F²/||M||_op²` is the noise-robust rank-proxy used in incoherence-bound diagnostics. **IDENTICAL to 259-M3, ship ONCE.**

**R3 — `rpca/balance.go::PCPBalance` ~30 LOC.** Universal-balance constants: `λ_PCP(m, n) = 1/sqrt(max(m, n))` per CLMW-2011 Thm 1.1; `λ_StablePCP(m, n, sigma_noise) = (1 + delta/sigma)/sqrt(max(m, n))` per Zhou-Li-Wright-Candes-Ma-2010 Thm 2. Centralised here so all RPCA solvers share the same default-tuning.

**R4 — `rpca/incoherence.go::IncoherenceParameter` ~20 LOC.** `IncoherenceParameter(L []float64, m, n, r int) float64` computes `μ(L) = max(max_i (n/r)·||U^T e_i||², max_j (m/r)·||V^T e_j||²)` where U,V from rank-r SVD of L. CLMW-2011 §A.1 recovery-guarantee certificate: PCP-recovery succeeds whp when μ(L) ≤ μ₀ for small μ₀ (incoherent low-rank component does not align with sparse-corruption support).

### Tier-2 — PCP / Inexact-ALM / Accelerated-ALM / Partial-ADMM (~480 LOC NEW)

**R5 — `rpca/pcp.go::PrincipalComponentPursuit` ~140 LOC — KEYSTONE.** `PCP(M []float64, m, n int, lambda float64, maxIter int) (L, S []float64)` solves `min ||L||_* + λ·||S||_1 s.t. M = L + S` via ADMM. Per-iter: L = D_{1/μ}(M − S + Y/μ); S = ProxL1(M − L + Y/μ; λ/μ); Y = Y + μ·(M − L − S). Default λ via R3 PCPBalance. **IDENTICAL to 215-C25 + 259-M17 — ship ONCE here, import from both 215-`cs/lowrank/` and 259-`mc/`.** Refs: Candes-Li-Ma-Wright-2011-JACM-58:11 *Robust principal component analysis?*.

**R6 — `rpca/inexact_alm.go::InexactALM` ~140 LOC — PRODUCTION-DEFAULT.** Lin-Chen-Ma-2010-arxiv-1009.5055 inexact-ALM solver. Approximate-minimisation per outer-iter via single-SVT-step + single-soft-threshold-step, dual-update with `μ_{k+1} = ρ·μ_k` for `ρ ≈ 1.1`. Avoids fully-converged inner-LS — cuts SVD calls 5-10x vs PCP-ADMM. Convergence certificate: stop when `||M − L − S||_F / ||M||_F ≤ 1e−7`. Refs: Lin-Chen-Ma-2010 §3.1.

**R7 — `rpca/accelerated_alm.go::AcceleratedALM` ~80 LOC.** Nesterov-momentum on top of R6 InexactALM. Maintains auxiliary `(L̂_k, Ŝ_k) = (L_k, S_k) + θ_k·((L_k, S_k) − (L_{k−1}, S_{k−1}))` with `θ_k = (k − 1)/(k + 2)` per Beck-Teboulle-2009-FISTA. ~2x faster than R6 on ill-conditioned RPCA.

**R8 — `rpca/partial_admm.go::PartialADMM` ~80 LOC.** Tao-Yuan-2011-SIOPT *Recovering low-rank and sparse components of matrices from incomplete and noisy observations* extends PCP to PARTIALLY-OBSERVED data: `min ||L||_* + λ·||S||_1 s.t. P_Ω(L + S) = P_Ω(M)`. The RPCA + matrix-completion fusion. Cross-link to 259-M6 ADMM-completion as the matrix-completion case where S = 0.

### Tier-3 — IRLS / Schatten-p / IRNN (~360 LOC NEW)

**R9 — `rpca/irls.go::IRLS-RPCA` ~140 LOC.** Iteratively-reweighted-least-squares Mohan-Fazel-2012-style for RPCA: `min Σ w_i^L·σ_i(L) + λ·Σ_j w_j^S·|S_j|` with weights `w_i^L = (σ_i² + ε)^{p/2−1}` for Schatten-p `0 < p < 1`, `w_j^S = (S_j² + ε)^{p/2−1}` for ℓ_p sparsity. ε-decreasing schedule. Sharper recovery than convex PCP for highly-low-rank cases. Refs: Mohan-Fazel-2012-JMLR-13:3441.

**R10 — `rpca/schatten_p.go::SchattenPRPCA` ~80 LOC.** Non-convex Schatten-p variant `min ||L||_{S^p}^p + λ·||S||_1 s.t. M = L + S` for `0 < p < 1` via singular-value-IRLS-thresholding. Refs: Gu-Bian-Liu-Ye-2014-ICML *Weighted nuclear norm minimization*.

**R11 — `rpca/irnn.go::IteratellyReweightedNuclearNorm` ~140 LOC.** Lu-Tang-Yan-Lin-2014-CVPR *Generalized nonconvex nonsmooth low-rank minimization*: at each outer iter solve weighted-nuclear-min `min Σ w_i·σ_i(L) + λ·||S||_1` with weights `w_i^{(k+1)} = ∂g(σ_i(L^{(k)}))` for surrogate g concave on positives — handles SCAD / MCP / log-sum / capped-l_1 / l_p uniformly via the surrogate-derivative interface.

### Tier-4 — GoDec / GreGoDec / Stable-PCP (~340 LOC NEW)

**R12 — `rpca/godec.go::GoDec` ~140 LOC.** Zhou-Tao-2011-ICML `min ||M − L − S||_F² s.t. rank(L) ≤ r, ||S||_0 ≤ k` via alternating-bilateral-random-projection — TOLERANT to dense-noise unlike PCP, ~5x faster via randomized-SVD. **IDENTICAL to 259-M16, ship ONCE.** Refs: Zhou-Tao-2011 *GoDec*.

**R13 — `rpca/gregodec.go::GreGoDec` ~80 LOC.** Zhou-Tao-2013 *Greedy bilateral sketch, completion & smoothing* — improves R12 convergence via greedy-rank-selection + smoothing-prior on L.

**R14 — `rpca/stable_pcp.go::StablePCP` ~120 LOC.** Zhou-Li-Wright-Candes-Ma-2010-ISIT `min ||L||_* + λ·||S||_1 s.t. ||M − L − S||_F ≤ δ` via ADMM-with-Frobenius-projection-step. The noise-tolerant RPCA variant for real-world background-subtraction where camera-sensor-noise is always present.

### Tier-5 — Online / streaming RPCA (~480 LOC NEW)

**R15 — `rpca/online_stochastic.go::OnlineStochasticRPCA` ~140 LOC — STREAMING-DEFAULT.** Feng-Xu-Yan-2013-NIPS *Online robust PCA via stochastic optimization*. `O(d·r)` memory, processes one column at a time via stochastic-gradient on dual-basis. Refs: Feng-Xu-Yan-2013-NIPS.

**R16 — `rpca/grasta.go::GRASTA` ~120 LOC — SOTA-STREAMING.** He-Balzano-Lui-2011-NIPS *Online robust subspace tracking from partial information*. Single rank-1 update per column-arrival on Grassmann(d, r) manifold, robust to sparse-corruption via L1-augmented-loss. Builds on `optim/manifold/Grassmann` (206-PR-A).

**R17 — `rpca/reprocs.go::ReProCS` ~120 LOC.** Guo-Qiu-Vaswani-2014-IT *An online algorithm for separating sparse and low-dimensional signal sequences from their sum* — recursive-projected-compressed-sensing for streaming RPCA when the sparse-component is structured per-frame. Used in video-foreground-extraction.

**R18 — `rpca/grouse_rpca.go::GROUSE-RPCA` ~100 LOC.** GROUSE-Recht-2010-Balzano-Nowak-Recht streaming-Grassmannian rank-r updates extended to handle sparse-corruption. Simplest possible online-RPCA, NO full-SVD per step. Cross-link to 259-M22 OnlineMC where the sparse-corruption term is dropped.

### Tier-6 — L1-PCA / M-estimator-PCA / Median-PCA (~480 LOC NEW)

**R19 — `rpca/l1_pca_subgrad.go::L1PCASubgrad` ~140 LOC.** Brooks-Dula-Boone-2013-CSDA `max_w ||X^T w||_1 s.t. ||w||_2 = 1` via sub-gradient-on-sphere. Robust to outlier-rows. Bypasses mean-subtraction outlier-sensitivity of classical PCA. Refs: Brooks-Dula-Boone-2013 *A pure L1-norm principal component analysis*.

**R20 — `rpca/l1_pca_exact.go::L1PCAExact` ~120 LOC — MOAT.** Markopoulos-Karystinos-Pados-2014-IEEE-TPAMI *Optimal algorithms for L1-subspace signal processing*. Combinatorial-search over `2^n` sign-patterns with O(n^{rk−k+1}) polynomial-time exact solver. GUARANTEED-OPTIMAL but only feasible for n ≤ 20. NO public Go implementation exists worldwide.

**R21 — `rpca/m_pca.go::MEstimatorPCA` ~140 LOC — CROSS-LINK-232.** Generic M-estimator-PCA `min Σ_i ρ(||x_i − P x_i||_2 / σ_i)` for ρ ∈ {Huber, Tukey-biweight, Cauchy, Welsch, Andrews, Hampel} via IRLS. **DIRECTLY composes against 232-R10 to R15 M-estimator-quartet shipped in `prob/robust/mestim/`** through the `MEstimator interface { Psi(r) float64; Weight(r) float64; Rho(r) float64; Tuning() float64 }` (see slot 232). The bridge between robust-statistics and robust-low-rank.

**R22 — `rpca/median_pca.go::MedianPCA` ~80 LOC.** Robust covariance via 232-R21 FastMCD → eigendecompose → robust-Mahalanobis projection. The simplest robust-PCA (closed-form covariance + standard eigen pipeline), 50% breakdown via MCD-covariance.

### Tier-7 — Tensor / Bayesian / Variational / Dictionary (~520 LOC NEW)

**R23 — `rpca/tensor_rpca.go::TensorRPCA` ~160 LOC — CROSS-LINK-257.** Lu-Feng-Yan-Lin-Yan-2016-CVPR *Tensor robust principal component analysis: Exact recovery of corrupted low-rank tensors via convex optimization*. Requires Gibbs-MCMC substrate (cross-link to 117 prob-MCMC).

**R25 — `rpca/variational.go::VariationalRPCA` ~120 LOC.** Babacan-Luessi-Molina-Katsaggelos-2012-IT *Sparse Bayesian methods for low-rank matrix estimation*. Variational-Bayesian RPCA via mean-field approximation. Faster than R24 Gibbs-MCMC; automatic-rank + automatic-sparsity-level.

**R26 — `rpca/dictionary_rpca.go::DictionaryRPCA` ~100 LOC — CROSS-LINK-258.** Sprechmann-Bronstein-Sapiro-2014 *Learning robust low-rank representations*.
