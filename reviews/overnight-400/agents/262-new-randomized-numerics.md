## 1. The 27-primitive surface N1-N27 (with composition)

Each entry: capability, composition over existing primitives, LOC, pin, and cross-link. Cross-references show ship-once primitives that satisfy multiple slots.

### Tier 1 — Sketch substrate (`linalg/sketch.go`, ~440 LOC)

**N1 `Sketch` interface** (~30 LOC). `Apply(A, m, n int, out []float64)` ; `ApplyT(A, m, n int, out []float64)` ; `Rows() int`. Polymorphic over Gaussian / Achlioptas / SparseJL / CountSketch / SRHT. Established interface idiom from 188-D-substrate, 224-ST1 Sketch[T], 225-N1 Index[T] — sixth interface in the unified-substrate keystone.

**N2 `GaussianSketch`** (~60 LOC). `S = (1/√m) · randn(m, n)`. Each entry N(0, 1/m). `Apply(A, n, k, out)` reduces to `MatMul(S, A, m, n, k, out)`. JL-lemma guarantee `||SAx||² ∈ (1±ε)||Ax||²` w.p. ≥ 1−δ for `m ≥ 4·ε⁻²·log(2/δ)`. **Composition:** `prob.StandardNormalBatch(m·n, rng, out) → out *= 1/√m`. The textbook starting point.

**N3 `AchlioptasSketch`** (~80 LOC). Achlioptas-2003-JCSS-66:671. Two variants: (a) ±1/√m Rademacher (s=1, dense), (b) {−√3, 0, +√3}/√m sparse (s=3, 2/3 zeros). Same JL-lemma constants as Gaussian but **3× memory** + **3× MatMul** savings via sparse matvec. **Production default** for in-memory sketches because IEEE-deterministic (no Box-Muller cache) and saves memory bandwidth. Reference: Achlioptas-2003 Theorem 1.1.

**N4 `SparseJLSketch`** (~120 LOC). Kane-Nelson-2014-J-ACM-61:4 OSNAP. `s` non-zeros per column (s=O(ε⁻¹·log(1/δ))). Each non-zero is ±1/√s in a hash-determined row. Apply runs in `nnz(A)·s` time. **Composition:** `crypto.MurmurHash3_32(col_id, seed_i)` for s i.i.d. row-hashes per column; `crypto.MurmurHash3_32(col_id, seed_i+1)` for sign. The substrate-keystone for input-sparsity-time sketching.

**N5 `CountSketch`** (~80 LOC). Charikar-Chen-Farach-Colton-2002-ICALP-29. The s=1 case of N4 — single hash per column + sign. Stored as `(h []int32, sigma []int8)` length-n arrays NOT a dense `m×n` matrix. **Apply runs in nnz(A) time**. Clarkson-Woodruff-2013-STOC-45 *Low-rank approximation in input sparsity time* uses CountSketch with `m = O(d²/ε²)` for `(1+ε)`-approximate-LS in `O(nnz(A) + d³/ε²)` flops. **CROSS-LINK to 224-ST6** ship-once architectural-home `linalg/sketch.go` (re-export wrapper from `streaming/`).

**N6 `SubsampledRandomizedHadamardTransform`** (~70 LOC). SRHT = `D·H·R` where D = diag(±1) random-sign, H = Walsh-Hadamard `n×n`, R = m random rows. Apply in `O(n·log(n))` via in-place butterfly (radix-2 dyadic transform). Tropp-2011-SIMAX-32:1141 proves m = `O(d·log·d / ε²)` suffices for JL with high probability. **Composition:** `signal/fft.go` butterfly scaffolding (Walsh-Hadamard is a sign-only FFT — same butterfly structure, no twiddles), random-sign D draws via `crypto.MurmurHash3_32` → bit, uniform-row-sample R via `combinatorics.RandomSubset`. The textbook fast-Johnson-Lindenstrauss.

### Tier 2 — Randomized-SVD core (`linalg/randomized.go`, ~580 LOC)

**N7 `RandomRangeFinder(matvec, n, k, p, q, rng) Q`** (~140 LOC). Halko-Martinsson-Tropp-2011 Algorithm-4.4. Composition: (1) draw n×(k+p) Ω via N2, (2) Y = Aᵠ·Ω with q-iteration `Y = (AAᵀ)ᵠ·A·Ω` for slow-decay spectra (q=1 default, q=2 slow-decay), (3) thin-QR on Y → Q. Returns column-orthonormal Q whose column-space approximates top-k right-singular subspace. Operator-error `||A - QQᵀA|| ≤ (1+α·√(min(m,n)/p))·σ_{k+1}` w.p. `1-3·p⁻ᵖ` for α=10. **CROSS-LINK to 188-D8 + 261-O20** ship-once.

**N8 `RandomizedSVD(matvec, n, k, p, q, rng) (U, S, V)`** (~180 LOC). Halko-Martinsson-Tropp-2011 Algorithm-5.1. (1) Q = N7, (2) B = QᵀA (k+p)×n, (3) small SVD(B) = Ũ Σ V via Golub-Reinsch (P0 substrate), (4) U = Q·Ũ. **The headline RandNLA primitive** — converts batch-PCA from O(n·d² + d³) to O(n·d·k + d·k²) for top-k. **CROSS-LINK to 188-D7 + 261-O5** architectural-canonical-home `linalg/randomized.go`.

**N9 `AdaptiveRandomizedSVD(matvec, n, p, q, ε, rng) (U, S, V)`** (~120 LOC). HMT-2011 Algorithm-4.2 a posteriori error-bound. Loop: grow k by 10 → run N7 → estimate `||A - QQᵀA||_F` via 10 random Frobenius probes → stop when ε reached. Auto-rank-detection without prior knowledge of k. **The production-default randomized-SVD** when k is unknown.

**N10 `SubspaceIteration(matvec, n, k, p, q, rng) Q`** (~80 LOC). Saad-1992 q-step subspace iteration. Used as power-method preconditioner inside N7 to amplify singular-gap when σ_k/σ_{k+1} ≈ 1.

**N11 `Nyström(K, n, k, p, rng) factors`** (~60 LOC). Williams-Seeger-2001 + Drineas-Mahoney-2005. SPD kernel-matrix approximation `K ≈ K(:,S)·K(S,S)⁺·K(S,:)` where S is k-column-subset (uniform / leverage-score / DPP). **CROSS-LINK to 236-RKHS + 237-GP** for K = kernel-Gram. Substrate for SparseGP (FITC, VFE) and KernelRidgeRegression.

### Tier 3 — Leverage-score + sampling (`linalg/sample.go`, ~360 LOC)

**N12 `LeverageScores(A, m, n, out)`** (~80 LOC). Drineas-Mahoney-Muthukrishnan-2006-SIAM-J-Comput-36:158. `ℓᵢ = ||(QᵀA)ᵢ||²` where `Q ∈ R^{m×r}` is column-orthonormal range. Σℓᵢ = r. The optimal importance-sampling probability for column-subset / coreset / CUR. Deterministic via thin-QR on A (blocked on 097-T1) or randomized via `Q = N7 RandomRangeFinder(A, k=r)`. **CROSS-LINK to 188-D11** ship-once.

**N13 `ApproximateLeverageScores(A, m, n, ε, rng, out)`** (~60 LOC). Drineas-Magdon-Ismail-Mahoney-Woodruff-2012-JMLR-13 *Fast approximation of matrix coherence and statistical leverage*. JL-sketched-pseudoinverse: ℓᵢ ≈ ||(SAR⁺)ᵢ||² for random sketches S, R. Time `O(nnz(A) + r³·ε⁻²·log·n)` vs `O(m·n·r)` exact. Production-default at scale.

**N14 `EffectiveResistance(L, n, edge_uv, ε, rng)`** (~80 LOC). Spielman-Srivastava-2011-SIAM-J-Comput-40:1913 *Graph sparsification by effective resistances*. R_uv = (e_u-e_v)ᵀ·L⁺·(e_u-e_v) on the graph-Laplacian. The graph-side equivalent of leverage scores: `ℓ_e = w_e · R_uv` is exactly the leverage score of edge e in the incidence-matrix least-squares system. **CROSS-LINK to graph/laplacian.go** which doesn't exist — would also need this for spectral-clustering / random-walk / heat-equation work.

**N15 `CURDecomposition(A, m, n, c, r, rng) (C, U, R)`** (~80 LOC). Mahoney-Drineas-2009-PNAS-106:697 *CUR matrix decompositions for improved data analysis*. C = c columns of A by leverage-score, R = r rows of A by leverage-score, U = `C⁺·A·R⁺`. Structurally-interpretable alternative to randomized-SVD: every row/column of CUR is a row/column of original A (preserves sparsity, names, units). For genomics / scientific data where interpretability matters more than orthogonal basis. **CROSS-LINK to 188-D12**.

### Tier 4 — Sketched-and-solve regression (`linalg/sketched_ls.go`, ~480 LOC)

**N16 `SketchedLeastSquares(A, m, n, b, ε, kind, rng) x`** (~180 LOC). Sarlós-2006-FOCS *Improved approximation algorithms for large matrices via random projections* + DMMS-2011-Numer-Math-117:219. Pick sketch dimension `d_S = O(d·log(d)/ε²)`. Apply S to A and b. Solve d_S × d small LS exactly via `linalg.LUSolve` on (SA)ᵀ(SA). Returns x* with `||Ax* - b|| ≤ (1+ε)·min_y ||Ay - b||` w.h.p. **CROSS-LINK to 188-D9** ship-once architectural-canonical-home `linalg/sketched_ls.go`.

**N17 `SketchedRidgeRegression(A, m, n, b, λ, ε, rng) x`** (~80 LOC). Tikhonov-regularised analogue of N16. Solve `min ||SAx - Sb||² + λ||x||²` exactly. Closes `prob/regression.go::Ridge` promise-debt against large-n. Used by 117-prob-missing T2 + 184-synergy-linalg-prob.

**N18 `IterativeSketchedLS(A, m, n, b, ε, rng) x`** (~120 LOC). Blendenpik (Avron-Maymounkov-Toledo-2010-SIMAX-32:1217) + LSRN (Meng-Mahoney-2013-STOC-45). Sketch-and-precondition: (1) sketch SA → small QR → preconditioner R⁻¹ such that AR⁻¹ has condition O(1), (2) run Krylov LSQR (or CG on normal equations) on `(AR⁻¹)·y = b, x = R⁻¹·y` to **machine precision**. Sketch-then-solve N16 gives O(ε)-error; sketch-and-precondition N18 gives ulp-error in O(d·log·d) Krylov-iterations. **The production-grade variant.**

**N19 `CompressedSensingLS(A, m, n, b, λ, rng) x`** (~100 LOC). Sparse-LS via sketched-FISTA. `min ||SAx - Sb||² + λ||x||₁`. Composes existing `optim/fista.go` (assumed present) with N4 SparseJLSketch sketch-step. **CROSS-LINK to 215-cs-package** for the compressed-sensing reconstruction primitive.

### Tier 5 — Randomized iterative solvers (`linalg/krylov.go`, ~480 LOC)

**N20 `RandomizedKaczmarz(A, m, n, b, ε, rng) x`** (~120 LOC). Strohmer-Vershynin-2009-J-Fourier-Anal-15:262. Importance-sample row i with prob `||Aᵢ||²/||A||_F²`, project current x onto hyperplane `{x : Aᵢ·x = bᵢ}`. Expected linear-convergence-rate `(1 - 1/κ_F²(A))^k`. Beats CGLS for highly-overdetermined LS (m ≫ n). **CROSS-LINK to 188-D17** ship-once.

**N21 `BlockRandomizedKaczmarz(A, m, n, blockSize, b, rng) x`** (~80 LOC). Needell-Tropp-2014-LAA-441:199. Pick block of rows (random sampling without replacement), project via small-LS solve (Cholesky of `AᵢAᵢᵀ`). Variance reduction over single-row Kaczmarz when rows are clustered.

**N22 `SketchAndProject(A, m, n, b, ε, rng) x`** (~80 LOC). Gower-Richtárik-2015-SIMAX-36:1660 unifying-framework. `x_{k+1} = argmin ||x - x_k||² s.t. SᵀAx = Sᵀb` for various S. Special cases: S = e_i → randomized coordinate descent; S = row-sample → randomized Kaczmarz; S = Newton-sketch → preconditioned Newton. The unifying meta-primitive.

**N23 `LanczosTridiag(matvec, n, k, rng) (alpha, beta, Q)`** (~120 LOC). Lanczos-1950 / Saad-2003-§6.6. Random-start k-step Krylov-tridiagonal `T_k = QᵀAQ`. Selective-reorthogonalization Simon-1984 for k≥30. **IDENTICAL to 188-D6 + 261-O19** ship-once architectural-canonical-home `linalg/krylov.go`. The substrate for N24 + truncated-SVD + spectral-density-estimation.

**N24 `RandomizedLanczosLogDet(matvec, n, k, m, rng) (logdet, stdErr)`** (~80 LOC). Ubaru-Chen-Saad-2017-SIMAX-38:1075 *Fast estimation of tr(f(A)) via stochastic Lanczos quadrature*. Estimate log-det = tr(log(A)) via m random Rademacher / Gaussian probes, each k-step Lanczos-tridiagonalised to T_k, log-det estimate `(1/m) Σ ||v||² · e₁ᵀ·log(T_k)·e₁` via small-matrix log-eigenvalue diagonalisation. **CROSS-LINK to 237-Gaussian-Process** marginal-likelihood `log p(y|θ) = -½·yᵀK⁻¹y - ½·log|K| - n/2·log(2π)` where the log|K| term is the bottleneck.

### Tier 6 — Streaming sketches as RandNLA primitives (`linalg/stream_sketch.go`, ~280 LOC)

**N25 `AMSF2Sketch`** (~120 LOC). Alon-Matias-Szegedy-1996-JCSS-58:137 second-frequency-moment-estimator. F₂ = Σ |xᵢ|² via `(1/(ε²·log(1/δ)))` Rademacher-projection-and-square. The streaming-norm-estimator. **CROSS-LINK to 224-ST10** ship-once `linalg/stream_sketch.go` re-export wrapper from `streaming/`.

**N26 `RandomizedMatMul(A, B, m, n, p, c, rng) out`** (~80 LOC). Drineas-Kannan-Mahoney-2006-SIAM-J-Comput-36:132 *Fast Monte Carlo algorithms for matrices*. Sample c columns of A (and corresponding rows of B) with prob `||Aᵢ||·||Bᵢ||/Σ`, sum outer-products. Expected `O(c·m·p)` flops vs `n·m·p`. **CROSS-LINK to 188-D10** ship-once.

**N27 `CountSketchTensor(A, B, T1, T2)`** (~80 LOC). Pham-Pagh-2013-KDD-19 + Avron-Nguyen-Woodruff-2014-NIPS *Subspace embeddings for the polynomial kernel*. TensorSketch via convolution-of-CountSketches: `CS(x ⊗ y) = CS_1(x) ⊛ CS_2(y)` (FFT-based convolution). The substrate for explicit-feature-map polynomial-kernel approximation. **CROSS-LINK to 257-D26 tensor-sketch + 236-RFF**.
