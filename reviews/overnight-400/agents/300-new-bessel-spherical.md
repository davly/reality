For non-integer ν, use Temme's series in `g_1(ν) = (1/(2ν))(1/Γ(1-ν) - 1/Γ(1+ν))` for small `z` and uniform asymptotic for large `z`. ~200 LOC. **Tier T5.**
- **Miller's algorithm (J.C.P. Miller 1952) is mandatory for J_n(z) integer order.** *Bessel Functions, Part II*, Cambridge — observation: `J_n(z)` upward recurrence `J_{n+1}(z) = (2n/z) J_n(z) - J_{n-1}(z)` is **catastrophically unstable** because `Y_n(z)` (the linearly independent solution) grows in the same direction. Miller's downward recurrence: start at large `N >> n` with arbitrary seed `J_N = 0, J_{N-1} = 1`; recur **downward** `J_{k-1} = (2k/z) J_k - J_{k+1}` (now Y_n is the *decaying* solution downward, J_n is the dominant — stable); rescale at the end using `Σ_k ε_k J_k(z) = 1` normalization (where ε_k = 2 for k≥1, ε_0 = 1) — this is the standard fix-up. Choice of starting `N`: rule-of-thumb `N = max(n + 20, n + 10·sqrt(n))` for ~14 digits. **The single most important numerical-Bessel fact, taught in every numerical-analysis textbook (Numerical Recipes §6.5, Press-Teukolsky-Vetterling-Flannery 2007, Olver-Maximon DLMF Ch.10).** ~80 LOC. **T1.**
- **Olver uniform asymptotic expansion** (Olver 1954, 1974 *Asymptotics and Special Functions*, A.K. Peters) for **large ν AND large z simultaneously** with `z/ν ≈ 1` (turning-point region where standard asymptotic in z fails). DLMF 10.20.4: `J_ν(νz) ~ (4ζ/(1-z²))^(1/4) [Ai(ν^(2/3) ζ) Σ A_k(ζ)/ν^(2k) + ν^(-4/3) Ai'(ν^(2/3) ζ) Σ B_k(ζ)/ν^(2k)]` where `ζ` is a transition variable. Requires Airy functions (`math.Airy` not in stdlib; ~80 LOC for Ai/Bi via series + asymptotic, separate primitive). **Tier T4** (production arbitrary-real-order). ~250 LOC over Airy. Day-3 frontier.
- **Spherical Bessel j_n, y_n, h_n^(1,2) — closed forms in elementary functions for integer n.** DLMF 10.49: `j_n(z) = (-z)^n (1/z d/dz)^n (sin(z)/z)`, `y_n(z) = -(-z)^n (1/z d/dz)^n (cos(z)/z)`. Concretely: `j_0(z) = sin(z)/z`, `j_1(z) = sin(z)/z² - cos(z)/z`, `y_0(z) = -cos(z)/z`, `y_1(z) = -cos(z)/z² - sin(z)/z`; integer n ≥ 2 by stable upward recurrence in `y_n` (dominant in upward direction) and Miller-downward in `j_n` (subdominant in upward direction). Or `j_n(z) = √(π/(2z)) J_{n+1/2}(z)` reducing to ordinary Bessel half-integer order. **~80 LOC for integer-n j_n/y_n.** **T2; day-1.**
- **Hankel functions H_ν^(1)(z) = J_ν(z) + i Y_ν(z), H_ν^(2)(z) = J_ν(z) - i Y_ν(z).** Trivial composition once J_ν, Y_ν exist. Used in **outgoing/incoming wave decomposition** (radiation Sommerfeld condition: outgoing wave = H^(1) for time-convention `e^(-iωt)`). **~40 LOC.** **T3; day-1.**
- **Riccati-Bessel S_n, C_n, ξ_n — Mie scattering primitives.** `S_n(z) = z j_n(z)`, `C_n(z) = -z y_n(z)`, `ξ_n(z) = z h_n^(1)(z) = S_n(z) + i χ_n(z)` where `χ_n(z) = -z y_n(z) = C_n(z)`. The Mie scattering coefficients for a sphere of complex refractive index `m` and size parameter `x = 2πa/λ` are: `a_n = [m ψ_n(mx) ψ'_n(x) - ψ_n(x) ψ'_n(mx)] / [m ψ_n(mx) ξ'_n(x) - ξ_n(x) ψ'_n(mx)]` where `ψ_n = S_n` (Riccati-Bessel, real) and `ξ_n` is the Riccati-Hankel; identical formula for `b_n` with `m` swapped between numerator/denominator. **~80 LOC over T2, T3.** **T7 + T8.** Used in Pistachio atmospheric scattering, slot 197 acoustics-fluids.
- **Kelvin functions ber, bei, ker, kei.** DLMF 10.61: `ber(x) + i·bei(x) = J_0(xe^(3iπ/4))`. Practically: power series in `(x/2)^(4k)` for both `ber` and `bei` with alternating sign pattern: `ber(x) = Σ_k (-1)^k (x/2)^(4k) / ((2k)!)²`, `bei(x) = Σ_k (-1)^k (x/2)^(4k+2) / ((2k+1)!)²`. `ker, kei` analogous from `K_0(xe^(iπ/4))`. Used in **AC skin effect in cylindrical conductors** (electrical engineering — the canonical EE reference is Olver-Maximon DLMF Ch.10.61). ~120 LOC. **T9.** Day-2.
