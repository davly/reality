### Tier-1 — ProxNuclear + nuclear/Schatten norm primitives (~120 LOC)

### Tier-2 — SVT / SVP / ADMM-completion / proximal-gradient (~440 LOC NEW)

**M4 — `mc/svt.go::SVT-iteration` ~140 LOC — KEYSTONE.** `SVT(observed map[[2]int]float64, m, n int, tau, beta float64, maxIter int) (X []float64)` iterates `X_{k+1} = D_τ(X_k − β·(P_Ω(X_k) − P_Ω(M)))`. Adaptive-step-size via `β_k = ||P_Ω(X_k) − P_Ω(M)||_F / ||X_k||_F` heuristic. Krylov-warm-start avoids full-SVD per iter via Lanczos-on-current-residual. Refs: Cai-Candes-Shen-2010 *A singular value thresholding algorithm for matrix completion* SIOPT-20:1956.

**M5 — `mc/svp.go::SVP` ~120 LOC.** `SVP(observed, m, n int, rank int, mu float64, maxIter int) (X []float64)` Singular-Value-Projection-Jain-Meka-Dhillon-2010: `X_{k+1} = H_r(X_k + μ·A^*(b − A(X_k)))` where H_r is truncated-SVD-keep-top-r. Provable RIP-MC convergence under δ_{2r} < 1/3. Refs: Jain-Meka-Dhillon-2010-NIPS *Guaranteed rank minimization via singular value projection*.

**M6 — `mc/admm.go::ADMM-completion` ~120 LOC.** `MatrixCompletionADMM(observed, m, n int, mu float64, maxIter int) (X []float64)` Lin-Chen-Ma-2010 ALM: `min ||L||_* s.t. P_Ω(L) = P_Ω(M)` via ADMM splitting. L-update = SVT(M − Y/μ; 1/μ); Y-update = dual-ascent. Faster convergence than SVT for ill-conditioned observations.

**M7 — `mc/proxgrad.go::ProximalGradient + FISTA-on-MC` ~60 LOC.** Wraps existing `optim/proximal/Fbs` and `optim/proximal/Fista` with `prox = ProxNuclear` and `gradFnc = P_Ω(X) − P_Ω(M)` for `min (1/2)||P_Ω(X − M)||_F^2 + τ·||X||_*`. The simplest possible MC primitive once M1 ships. Refs: Toh-Yun-2010 *An accelerated proximal gradient algorithm for nuclear norm regularized linear least squares problems*.

### Tier-3 — IRLS + Schatten-p + ALS (~440 LOC NEW)

**M8 — `mc/irls.go::IRLSrank` ~140 LOC.** `IRLSRank(observed, m, n int, rank int, p, eps float64, maxIter int) (X []float64)` Mohan-Fazel-2012 iteratively-reweighted-least-squares for Schatten-p-norm. At each iter solve weighted-LS with weights `W_k = (X_k^T X_k + ε·I)^{p/2−1}`. ε-decreasing schedule for non-convex p < 1 sharper-than-nuclear sparsity. Refs: Mohan-Fazel-2012-JMLR-13:3441 *Iterative reweighted algorithms for matrix rank minimization*.

**M9 — `mc/schatten_p.go::SchattenPProximal` ~80 LOC.** `ProxSchattenP(M, m, n int, p, gamma float64, out)` for `0 < p ≤ 1` via singular-value-IRLS-thresholding. p=1 collapses to ProxNuclear; p=0 collapses to ProxNuclearTopR.

**M10 — `mc/als.go::AlternatingLeastSquares` ~120 LOC — KEYSTONE.** `ALS(observed, m, n int, rank int, maxIter int) (U, V [][]float64)` solves `min_{U ∈ R^{m×r}, V ∈ R^{n×r}} ||P_Ω(M − UV^T)||_F^2`. U-step: row-wise LS solve on `[V^T_{Ω_i}; ...] u_i = m_i`; V-step: column-wise LS solve. Provable convergence under RIP-MC + two-stage init via SVT-warm-start. Refs: Jain-Netrapalli-Sanghavi-2013-STOC-45:665 *Low-rank matrix completion using alternating minimization*.

**M11 — `mc/als_regularized.go::ALSRegularized` ~100 LOC.** `ALSRidge(observed, m, n int, rank int, lambdaU, lambdaV float64, maxIter int)` Funk-2006 Netflix-prize ridge variant `min ||P_Ω(M − UV^T)||_F^2 + λ_U·||U||_F^2 + λ_V·||V||_F^2`. Closed-form per-row update via `(V^T_Ω V_Ω + λ_U·I)^{-1} V^T_Ω m`.

### Tier-4 — Riemannian rank-r manifold (~280 LOC NEW additive on 206-R)

**M12 — `optim/manifold/fixedrank.go::FixedRankManifold` ~140 LOC.** Concrete 206-`Manifold` instance for M_r = {X ∈ R^{m×n} : rank(X) = r}. Tangent-space `T_X M_r = {U·A·V^T + U·B^T + C·V^T : A ∈ R^{r×r}, B ∈ R^{n×r}, C ∈ R^{m×r}, U^T·C = 0, V^T·B = 0}`. Retraction `R_X(ξ) = TruncatedSVD_r(X + ξ)` uses 215-PR-B SVD substrate. Vector-transport via `T_{Y←X}(ξ) = P_{T_Y}(ξ)`. Refs: Vandereycken-2013-SIOPT-23:1214 §2; Absil-Mahony-Sepulchre-2008 *Optimization Algorithms on Matrix Manifolds* Ch.7.

**M13 — `mc/rtrmc.go::RTRMC` ~70 LOC.** Riemannian-Trust-Region-MC-Boumal-Absil-2011-NIPS using 206-R3 Riemannian-Newton + M12 FixedRankManifold. Inner trust-region solver via 206-R-conjugate-gradient. Refs: Boumal-Absil-2011-NIPS *RTRMC: A Riemannian trust-region method for low-rank matrix completion*.

**M14 — `mc/lrgeomcg.go::LRGeomCG` ~70 LOC — SOTA-2013-2026.** Riemannian-Conjugate-Gradient on M_r with vector-transport-via-projection. ~10x faster than SVT for sparse-matrix-completion `n > 1000` per Vandereycken-2013 Fig 3. Default-recommended Riemannian-MC solver. Refs: Vandereycken-2013-SIOPT-23:1214 *Low-rank matrix completion by Riemannian optimization*.

### Tier-5 — OptSpace / GoDec / RPCA (~360 LOC NEW)

**M15 — `mc/optspace.go::OptSpace` ~140 LOC.** Keshavan-Montanari-Oh-2010-IT-56:2980 trim-then-SVD-init + gradient-on-Grassmannian-refinement. EARLIEST proven-optimal MC algorithm. Information-theoretic-optimal sample-complexity `m ≥ C·n·r·log(n)` matching Candes-Tao lower bound (Candes-Recht-2009 was `n^{1.2}r·log(n)`, Recht-2011 improved to `nr·log²(n)`, OptSpace already achieves `nr·log(n)` since 2010). Refs: Keshavan-Montanari-Oh-2010 *Matrix completion from a few entries*.

**M16 — `mc/godec.go::GoDec` ~120 LOC.** Zhou-Tao-2011-ICML randomized low-rank+sparse decomposition `min ||M − L − S||_F² s.t. rank(L) ≤ r, ||S||_0 ≤ k` via alternating bilateral random projections. Beats RPCA on noisy data and ~5x faster via randomized-SVD substitution. Refs: Zhou-Tao-2011-ICML *GoDec: Randomized low-rank & sparse matrix decomposition in noisy case*.

**M17 — `mc/rpca.go::RobustPCA` ~100 LOC.** Principal-Component-Pursuit-Candes-Li-Ma-Wright-2011 `min ||L||_* + λ·||S||_1 s.t. M = L + S` via ADMM-SVT-soft-threshold-alternation. λ = 1/sqrt(max(m,n)) the universal-balance per CLMW-2011 Thm 1.1. **Identical primitive to 215-C25 — ship ONCE here, import from 215-`cs/lowrank/`.** Refs: Candes-Li-Ma-Wright-2011-JACM-58:11 *Robust principal component analysis?*.

### Tier-6 — PMF + Bayesian + 1-bit MC (~440 LOC NEW)

**M18 — `mc/pmf.go::ProbabilisticMatrixFactorization` ~140 LOC.** Salakhutdinov-Mnih-2007-NIPS graphical-model `P(M_{ij} | U_i, V_j) = N(U_i^T V_j, σ²) × P(U) = N(0, σ_U² I) × P(V) = N(0, σ_V² I)` solved by SGD-MAP. Netflix-prize-2009-winner substrate. Refs: Salakhutdinov-Mnih-2007-NIPS-21 *Probabilistic matrix factorization*.

**M19 — `mc/bpmf.go::BayesianPMF` ~140 LOC — SOTA-Bayesian-2008-2026.** Fully-Bayesian-posterior over (U, V) via Gibbs-sampling on conjugate Normal-Wishart hyperprior. Outperforms PMF on Netflix without overfitting. Refs: Salakhutdinov-Mnih-2008-ICML *Bayesian probabilistic matrix factorization using Markov chain Monte Carlo*.

**M20 — `mc/onebit.go::OneBitMC` ~160 LOC — FRONTIER.** Davenport-Plan-vandenBerg-Wootters-2014 1-bit MC: recover M from Bernoulli observations `Y_{ij} ~ Bern(f(M_{ij}))` via nuclear-norm + max-norm constraints with logistic-link f. Extends MC to BINARY observations. The FIRST MC method for non-Gaussian observations. Refs: Davenport-Plan-vandenBerg-Wootters-2014-IT-60:2877 *1-bit matrix completion*.

**M21 — `mc/exp_family.go::ExpFamilyMC` ~80 LOC.** Lafond-2015-IT-61:5481 generalised exp-family-link MC unifying Gaussian / Bernoulli / Poisson / multinomial under one loss. Subsumes M20 1-bit MC.

### Tier-7 — online + coupled + advanced (~340 LOC NEW)

**M22 — `mc/online.go::OnlineMC` ~120 LOC.** GROUSE-Recht-2010 / Brand-2002 streaming-update on Grassmannian. When each column arrives, single rank-1 perturbation update U ∈ Grassmann(n,r). No full SVD per step. The streaming-recommendation-system substrate.

**M23 — `mc/coupled.go::CoupledMC` ~100 LOC.** Singh-Gordon-2008-KDD simultaneous completion of two matrices M and N sharing one factor `M ≈ U V^T, N ≈ U W^T`. Multi-view collaborative-filtering substrate. Refs: Singh-Gordon-2008-KDD *Relational learning via collective matrix factorization*.

**M24 — `mc/quaternion.go::QuaternionMC` ~80 LOC.** Color-image-completion via quaternion-matrix-MC. RGB pixel = pure-quaternion (i, j, k components). Embed to 4n×4n real matrix and run nuclear-norm. Refs: Jia-Liu-Vong-Ng-2019-CVPR *Robust quaternion matrix completion*.

**M25 — `mc/phase_retrieval.go::PhaseLift` ~60 LOC.** Phase-retrieval-as-rank-1-MC: lift x ∈ C^n to X = xx^T ∈ C^{n×n} rank-1, recover X from `tr(a_k a_k^T X) = b_k = |a_k^T x|²` via `min ||X||_* s.t. tr(a_k a_k^T X) = b_k`. **Identical to 215-T6 PhaseLift; ship ONCE here, import from 215-`cs/phaseret/`.** Refs: Candes-Strohmer-Voroninski-2013-CPAM-66:1241 *PhaseLift*.

**M26 — `mc/multi_task.go::MultiTaskMC` ~80 LOC.** Pong-Tseng-Ji-Ye-2010 simultaneous regularization across multiple matrices via stacked-nuclear-norm `min Σ_k ||M_k − L_k||_F² + λ·||[L_1, ..., L_K]||_*` for transfer-learning across recommendation tasks.
